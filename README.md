# Descrição do projeto
-> O FilePriv é um protótipo de um sistema distribuído com criptografia. 

# Descrição das funcionalidades
-> É possível criar usuários, fazer o upload e download de arquivos. 

# Instruções de instalação
-> Instale as seguintes tecnologias e ferramentas:
-> Use em uma máquina linux, de preferência baseada no debian. 
-> VirtualBox, Vagrant, Node.js.
