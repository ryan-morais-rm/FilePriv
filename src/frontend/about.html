Description

FilePriv is a functional prototype of a distributed secure storage server. Users can upload and download files with strong encryption and partitioning.

Files are encrypted on the bit level and split into independent parts, each protected with a unique private key to ensure maximum security.

Features & Technologies

  • Upload & Download: Send and retrieve encrypted files via the web interface.
  • Encryption: Secure AES-GCM encryption per file.
  • Random Keys: Unique key generation for every file upload.
  • Partition Protection: Each partition secured individually.
  • Reconstruction: Recover full files using password-based reconstruction.
